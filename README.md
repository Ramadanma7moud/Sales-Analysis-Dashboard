# Executive Sales Analysis Dashboard

An interactive sales analytics dashboard built entirely in Microsoft Excel, designed to give decision-makers a clear, real-time view of business performance.

## 🔑 Key Features
- KPI cards: Total Sales, Profit, Cost, and Quantity
- Sales & Profit trend analysis (2014–2017)
- Sales breakdown by Ship Mode
- Sub-Category profit analysis
- Top 10 states by sales
- Interactive slicers: Category, Region, Ship Date, Years

## 🛠️ Tools Used
Microsoft Excel — PivotTables, PivotCharts, Slicers, Dashboard Design

## 📊 Purpose
Transform raw sales data into actionable insights supporting decisions on pricing, shipping strategy, and geographic sales distribution.
